## El Origen
Monogram nace de la necesidad de **cambiar la forma de aprendizaje**, de explorar nuevas herramientas y una forma de tener un registro o "bitácora" para el aprendizaje propio

## El Setup Inicial
1. **OS:** Ubuntu 24.04 LTS. Linux
2. **Herramientas:** Astro Nodejs Git/Github

# ¿Por que?
Astro y Nodejs son frameworks ideales para proyectos pequeños como este portafolio por su velocidad y adaptabilidad (Aún estoy aprendiendo)

# ¿Cómo lo hice?
1. **Ejecutar:** npm create astro@latest monogram
2. ** Selecciona una plantilla**  
3. **Inicializar el repositorio**
4. **Crear la estructura de las carpetas dentro de la plantilla:**
cd monogram
mkdir -p src/content/academic src/content/personal**
5. Y así nace nuestro primer mono (post)
